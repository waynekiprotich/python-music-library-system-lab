# Music Library System
Overview
This project implements a Song class using Object-Oriented Programming in Python.
The system tracks individual song details while also maintaining global statistics about all songs created.

 
# Features
Create a Song with:
name
artist
genre
Automatically tracks:
  Total number of songs (Song.count)
  Unique artists (Song.artists)
  Unique genres (Song.genres)
  Number of songs per genre (Song.genre_count)
  Number of songs per artist (Song.artists_count)

# Concepts Used
Class attributes
Class methods
Dictionaries for counting
Object-Oriented Programming (OOP)
